Tv N1 for XBMC
========================

### Summary ###
This is a plugin for [XBMC](http://xbmc.org) that enables watching videos and
the live stream from <http://rs.n1info.com>.

### Setup/Installation ###
The plugin is available in LazyRobot addon repository.
