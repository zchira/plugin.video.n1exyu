About

This is Kodi plugin for N1-Info regional tv station. N1-Info covers ex-Yu region (Serbia, Croatia and BiH). The plugin allows you to watch live streams for any of three regions and some tv shows.

This isn't official plugin and I am not in any connection with TV N1. I made it for my personal use, because my cable tv operator doesn't offer N1-Info channel.

How to install

This plugin (and some other) may be installed from LazyRobot Kodi plugin repository. Download repository archive and instal it from Kodi (Video->Add on->Install from Zip).
